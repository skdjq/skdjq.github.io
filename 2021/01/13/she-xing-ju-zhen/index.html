蛇形矩阵

题目地址：https://www.acwing.com/problem/content/758/

输入两个整数n和m，输出一个n行m列的矩阵，将数字 1 到 n*m 按照回字蛇形填充至矩阵中。

具体矩阵形式可参考样例。

输入格式

输入共一行，包含两个整数n和m。

输出格式

输出满足要求的矩阵。

矩阵占n行，每行包含m个空格隔开的整数。

数据范围

1 ≤ n,m ≤ 1001 ≤ n,m ≤ 100

输入样例：

3 3

输出样例：

1 2 3
8 9 4
7 6 5

对于这一类问题，我们可能首先会想到去套很多个循环解决，但是这样代码量是非常大的，偏移法可以大大简化这一步骤。

什么是偏移法？

顾名思义，无论朝哪个方向走，我们把每次要走的位移偏差记录下来，存进数组中，这样会大大减少我们的代码量。

看图说话：

每当我们走到边界，d就会改变一次状态，那么什么是边界呢？

边界分两种情况：

  • 即将要走的那一步的下标超出数组下标，也就是越界。
  • 即将要走的那一步已经走过了。

知道了上面这几点，直接看代码：

import java.util.*;

public class Main{
    public static void main(String[] args) {
        Scanner scan = new Scanner(System.in);
        int n = scan.nextInt();
        int m = scan.nextInt();
        int[][] arr = new int[n][m];
        int[] dx = {-1, 0, 1, 0};
        int[] dy = {0, 1, 0, -1};
        int x = 0, y = 0;
        int d = 1;
        for(int i = 1; i <= n * m; i++) {
            arr[x][y] = i;
            // 计算下面要走的一步是否碰到边界
            int a = x + dx[d];
            int b = y + dy[d];
            if(a < 0 || a >= n || b < 0 || b >= m || arr[a][b] != 0) {
                // 如果碰到边界，改变状态
                d = (d + 1) % 4;
                a = x + dx[d];
                b = y + dy[d];
            }
            x = a;
            y = b;
        }
        for(int i = 0; i < n; i++) {
            for(int j = 0; j < m; j++) {
                System.out.print(arr[i][j] + " ");
            }
            System.out.println();
        }
    }
}

对于偏移数组，我们可以尝试去记住，对于这类问题我们都可以用偏移数组去解决，以简化我们的代码。
