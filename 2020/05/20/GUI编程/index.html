组件和容器

1.Frame

创建Frame对象

Frame frame = new Frame("这是我的第一个窗口");//内容为标签内容

一些方法：

frame.setVisible(boolean b);        //设置窗口可见性
frame.setBackground(Color color);    //设置背景颜色
frame.setBounds(int x, int y, int width, int height);        //设置起点坐标（（0,0）在左上方）和宽高
frame.setLayout();                    //设置布局
frame.setResizable(boolean b);        //设置是否可以调节窗口大小

自定义创建myFrame: 

class MyFrame extends Frame {
    static int count = 0;
    public MyFrame(int x, int y, int h, int w, Color color) {
        super("skdjq思考的机器" +(++count));
        setVisible(true);
        setBounds(x, y, w, h);
        setBackground(color);
    }
}

2.Panel

  • Panel 可以看成一个空间，但是不能单独存在。

  • Panel的许多方法和Frame相同。

  • Panel的坐标是相对于窗口Frame的坐标。

逻辑实例：

public class demo2 {
    public static void main(String[] args) {
        Frame frame = new Frame("GUI Test");
        Panel panel = new Panel();
        //设置布局
        frame.setLayout(null);
        frame.setBounds(300,300,500,500);
        frame.setBackground(new Color(100,100,100));

        panel.setBounds(100, 100, 400, 400);
        panel.setBackground(new Color(200,200,200));
        //添加 Panel 
        frame.add(panel);
        frame.setVisible(true);    
        //监听窗口关闭事件
        frame.addWindowListener(new WindowAdapter(){
            //窗口点击关闭需要做的事情
            public void windowClosing(WindowEvent e){
                System.exit(0);
            }
        });
    }
}

Frame 的add()方法可以添加Panel

3.布局管理器

  • 流式布局 FlowLayout

特点：从左到右的顺序排列，默认居中。

public class demo3 {
    public static void main(String[] args) {
        Frame frame = new Frame();
        Button button1 = new Button("Hello");
        Button button2 = new Button("World");
        Button button3 = new Button("Java");
        //设置为流式布局
        frame.setLayout(new FlowLayout(FlowLayout.LEFT));
        frame.setBounds(100, 100, 500, 500);
        frame.setBackground(new Color(200,200,200));
        frame.add(button1);
        frame.add(button2);
        frame.add(button3);
        frame.setVisible(true);
    }
}
  • 东西南北中 Borderlayout

特点：按照东南西北中的顺序排列， Frame默认的布局管理器。

其中每个部分只可添加一个组件，若添加多个组件，则最后添加的组件会覆盖前面的。

public class demo4 {
    public static void main(String[] args) {
        Frame frame = new Frame();
        Button b1 = new Button("button1");
        Button b2 = new Button("button2");
        Button b3 = new Button("button3");
        Button b4 = new Button("button4");
        Button b5 = new Button("button5");
        //添加按钮，通过 BorderLayout.xxx 设置布局位置
        frame.add(b1,BorderLayout.EAST);
        frame.add(b2,BorderLayout.WEST);
        frame.add(b3,BorderLayout.SOUTH);
        frame.add(b4,BorderLayout.NORTH);
        frame.add(b5,BorderLayout.CENTER);
        frame.setVisible(true);
        frame.setBounds(100,100,500,500);
    }
}
  • 表格布局 GridLayout

特点：规则的矩阵 ，系统自带的计算器用的就是这个布局。

public class demo5 {
    public static void main(String[] args) {
        Frame frame = new Frame();
        Button b1 = new Button("button1");
        Button b2 = new Button("button2");
        Button b3 = new Button("button3");
        Button b4 = new Button("button4");
        Button b5 = new Button("button5");
        Button b6 = new Button("button6");
        //设置为3行2列的网格式布局
        frame.setLayout(new GridLayout(3,2));
        frame.add(b1);
        frame.add(b2);
        frame.add(b3);
        frame.add(b4);
        frame.add(b5);
        frame.add(b6);
        frame.setVisible(true);
    }
}
