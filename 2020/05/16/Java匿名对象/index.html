匿名对象

我们知道new一个对象要有四个步骤：

  • 在堆上开辟一块新的空间，初始化所有值
  • 初始化显式设定的值
  • 调用构造函数
  • 返回该内存的地址给变量

匿名对象是指在创建对象时，只有创建对象的语句，却没有把对象的地址赋给某个变量。如：

//创建一个普通对象
Person p = new Person();
//创建一个匿名对象
new Person();

匿名对象的特点

  • 创建匿名对象可以直接使用，没有变量名。

    new Person().work();     //work 方法被一个没有名字的 Person 对象调用了

  • 匿名对象在没有指定其引用变量时，只能使用一次。

    new Person().work();     //创建一个匿名对象，调用 work 方法
new Person().work();     //想再次调用 work 方法，要重新创建一个匿名对象

  • 匿名对象可以作为方法接收的参数，方法返回值使用。如：

    return new Person();
