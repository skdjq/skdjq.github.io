通讯录管理系统

项目介绍

此项目管理系统采用C++编写，是一个很好的初学者练习项目，能够深化初学者对数组、结构体和指针的理解。

一般而言，在学习完指针、数组和结构体后就可以尝试练习编写该项目。

此通讯管理系统一共有六个功能：

1、添加联系人

2、显示联系人

3、删除联系人

4、查找联系人

5、修改联系人

6、清空联系人

初始化变量

初始化结构体

首先我们要定义一个通讯录成员结构体，用来存放存入通讯录人员的具体信息：

struct Person
{
    string name;        //姓名 
    int sex;            //性别 
    int age;            //年龄 
    string pNumber;        //电话 
    string Addr;        //住址 
};

然后我们定义一个存放人员的通讯录结构体：

#define MAX 1000                //方便对数组存放的最大值进行更改
struct addressbooks
{
    struct Person arr[MAX];        //通讯录最大容纳人数 
    int num;                    //通讯录中人员个数 
};

这样，我们的两个结构体就定义完成了。

初始化菜单

void Menu()
{
    cout << "***************************" << endl;
    cout << "*****  1、添加联系人  *****" << endl;
    cout << "*****  2、显示联系人  *****" << endl;
    cout << "*****  3、删除联系人  *****" << endl;
    cout << "*****  4、查找联系人  *****" << endl;
    cout << "*****  5、修改联系人  *****" << endl;
    cout << "*****  6、清空联系人  *****" << endl;
    cout << "*****  0、退出联系人  *****" << endl;
    cout << "***************************" << endl;
}

此为我们的通讯录菜单。

一、添加联系人

由于我们在改变形参时同时需要影响实参，所以我们采用的传递方式是地址传递。

由于我们是采用函数封装的形式，所以在函数定义时必须有形参，而这个形参就是用我们定义的struct addressbooks结构体类型定义的指针：struct addressbooks * abs。

tips：由struct addressbooks结构体类型定义的指针能够访问我们定义的结构体内的成员变量，并且更加节省内存空间。

我们在添加联系人时要分两种情况：

一是结构体数组内达到了最大值MAX，那么此时我们就不能够再添加联系人进去了。

二是数组内仍有空间，此时我们再进行添加的操作。

那么基于上述两种情况，添加联系人的函数代码为：

void addPerson (struct addressbooks * abs)
{
    string name;        //姓名 
    int sex;            //性别 
    int age;            //年龄 
    string pNumber;        //电话 
    string Addr;        //住址 

    if (abs->num == MAX)
    {
        cout << "通讯录人数已满，请勿添加！" << endl;
        return;
    }
    else
    {
        cout << "请添加联系人姓名：" << endl;
        cin >> name;
        abs->arr[abs->num].name = name;

        cout << "性别为：" << endl; 
        cout << "\"1\" --- 男" <> sex;
            if (sex == 1 || sex == 0)
            {
                abs->arr[abs->num].sex = sex;
                break;                //break 不能终止 if 循环 
            }
            else cout <<"输入有误！请重新输入！" << endl;
        }

        cout << "请输入联系人年龄：" << endl;
        cin >> age;
        abs->arr[abs->num].age = age;

        cout << "请输入联系人电话：" << endl;
        cin >> pNumber;
        abs->arr[abs->num].pNumber = pNumber;

        cout << "请输入联系人家庭住址：" << endl;
        cin >> Addr;
        abs->arr[abs->num].Addr = Addr;

        abs->num ++;
        cout << "添加成功!!" << endl;
        system ("pause");        //请按任意键继续 
        system ("cls");            //清屏 
    }
}

值得注意的是：abs是我们定义的能够存放struct addressbooks类型地址的指针变量，由指针访问结构体成员时，我们用->而不是.，比如abs->arr[abs->num]，而通过除了指针以外的方式访问结构体成员时，我们用.，比如arr[i].name。

在上述代码中我们可以很清晰的理解abs->arr[abs->num].name的含义：通过指针访问了struct addressbooks内的arr[]，并且访问的是arr[]下标为num的元素，由于num本身也属于struct addressbooks的成员，因此我们需要再次使用指针访问num，即abs->num，由于arr[]是由struct Person创建的，所以我们通过.可以访问struct Person内的成员变量，即arr[abs->num].name，综上我们就能很好的理解abs->arr[abs->num].name的含义了。

二、显示联系人

在有了上面我们对指针和数组访问结构体成员变量的理解之后，显示联系人就不难掌握了。

我们同样分两种情况：

一是通讯录还没有存入任何联系人，这时我们显示通讯录为空。

二是通讯录已经存入一个及以上的联系人了，我们就把他们的信息全部显示出来。

void showMenu(struct addressbooks * abs)
{
    int i ;
    if (abs->num == 0)
    {
        cout << "联系人目录为空！" << endl;
        system ("pause");
        system ("cls");
    }
    else
    {
        for (i = 0;inum;i++)
        {
            cout << abs->arr[i].name << "\t";
            if(abs->arr[i].sex == 1)
            {
                cout << "男" << "\t"; 
            }
            else
            {
                cout << "女" << "\t";
            }
            cout << abs->arr[i].age << "\t";
            cout << abs->arr[i].pNumber << "\t";
            cout << abs->arr[i].Addr << "\t" << endl;

        } 
        system("pause");
        system("cls");
    }
}

三、检查是否有相同的联系人

为什么我们要单独把这一块拿出来单讲呢？因为在后面的删除、查找、修改联系人中，我们都需要先判别是否通讯录已经有这一联系人存在，若压根没有这个联系人，那还删除、查找、修改不就是扯蛋吗。

因此，我们必须先检查是否有目标联系人存在于通讯录内。

int checkPerson(struct addressbooks * abs,string name)
{
    int i;
    for (i = 0;inum;i++)
    {
        if (abs->arr[i].name == name)
        {
            return i;
        }
    }
    return -1;
}

我们采用的是遍历整个数组，来寻找有无姓名相同的联系人，若有则返回该联系人所在位置的下标，方便我们后续对其进行操作。（这种检查方式不够严密，原因在于如果有重名的联系人系统就只会作用于先存入通讯录的那一个，读者如果有心，可以把电话号码也作为一项查找的标准，增加检查的精度。）

删除联系人

由于这个项目并没有采用链表的形式而使用了数组来存储数据，因此在删除联系人时可能没有那么方便，在这里我们使用的是前移法。即从需要删除的那一个元素起，数组内后一个元素的值向前移一位，这样就可以把前一位元素的值覆盖掉，也就是实现了删除的功能。这么说可能有点晦涩，我们来看代码：

void deletePerson(struct addressbooks * abs)
{
    cout << "请输入您要删除的联系人姓名：" << endl;
    string name;
    cin >> name;
    int a = checkPerson(abs,name);
    if (a != -1)
    {
        for (int i = a;i < abs->num;i++)
        {
            abs->arr[i] = abs->arr[i + 1];                //数组删除一个元素：从被删除项起，后面每一个元素向前移一位，覆盖掉前一个数。 
            cout << "删除成功！" << endl;
        }
        abs->num --;
    }
    else
    {
        cout << "查无此人!" << endl;
    }
    system ("pause");
    system ("cls");
}

这里我们千万不能忘记在删除一个联系人后，我们的num值要减一。

这样就一目了然了！如果还有不懂的地方，只需要画一个图就能理解啦！

查找联系人

这里没有什么特别难的，和上面一样，通过checkPerson函数查找有没有这个人，如果有，就打印出这个人的全部信息，没有则输出查无此人。

void findPerson(struct addressbooks * abs)
{    
    cout << "请输入您要查找的联系人的姓名：" << endl;
    string name;
    cin >> name;
    int ret = checkPerson(abs,name);
    if (ret != -1)
    {
        cout << abs->arr[ret].name << "\t";
        cout << abs->arr[ret].sex << "\t";
        cout << abs->arr[ret].age << "\t";
        cout << abs->arr[ret].pNumber << "\t";
        cout << abs->arr[ret].Addr << "\t";
    }
    else
    {
        cout << "查无此人！！" << endl;
    }
    system ("pause");
    system ("cls");
}

修改联系人

这个看起来比较难，实际上并没有新的东西，我们只需要通过checkPerson函数来判断有无我们要修改的这个人，，如果有，再采用和addPerson里几乎一样的方法重新修改这个人的信息即可。

这里我们需要注意的是，修改后千万不能使num的值改变，因为是修改，元素的总个数不会改变。

void modifyPerson(struct addressbooks * abs)
{
    int sex,age;
    string pNumber,Addr,name1;
    cout << "请输入您要修改的联系人姓名：" << endl;
    string name;
    cin >> name;
    int ret =  checkPerson(abs,name);
    if (ret != -1)
    {
        cout << "请添加联系人姓名：" << endl;
        cin >> name1;
        abs->arr[ret].name = name1;

        cout << "性别为：" << endl; 
        cout << "\"1\" --- 男" <> sex;
            if (sex == 1 || sex == 0)
            {
                abs->arr[ret].sex = sex;
                break;                //break 不能终止 if 循环 
            }
            else cout <<"输入有误！请重新输入！" << endl;
        }

        cout << "请输入联系人年龄：" << endl;
        cin >> age;
        abs->arr[ret].age = age;

        cout << "请输入联系人电话：" << endl;
        cin >> pNumber;
        abs->arr[ret].pNumber = pNumber;

        cout << "请输入联系人家庭住址：" << endl;
        cin >> Addr;
        abs->arr[ret].Addr = Addr;

        cout << "修改成功!!" << endl;
    }
    else
    {
        cout << "查无此人" << endl;
    }

清空联系人

这个就非常简单了，我们在清空联系人时并不需要实际上把每一个联系人都一个一个删掉，我们只需要实现逻辑上的清零，让控制台打印不出来那些联系人即可。

void cleanAddressbooks(struct addressbooks * abs)
{
    abs->num = 0;
    cout << "通讯录已被清空！" << endl;
    system("pause");
    system("cls");
}

正如上面的代码，我们只需要更改num为0即可实现逻辑上的清零。

主函数

讲了这么多，终于到我们的主函数了。由于我们的通讯录是有多个功能，每按下一个键就实现一个功能，这里我们想到了什么呢？没错！switch函数！我们只需要使用switch函数就能很好解决按一个键实现一个功能的效果了。

int main ()
{
    int select;
    struct addressbooks abs;
    abs.num = 0;
    while (1)
    {
        Menu();
        cin >> select;
        switch (select)    
        {
            case 1:
                addPerson(&abs);
                break;
            case 2:
                showMenu(&abs); 
                break;
            case 3:
                deletePerson(&abs);
                break;
            case 4:
                findPerson(&abs);
                break;
            case 5:
                modifyPerson(&abs);
                break;
            case 6:
                cleanAddressbooks(&abs);
                break;
            case 0:
                cout << "欢迎您下次再次使用通讯录！" << endl;
                system ("pause");
                return 0;
                break;
            default :
                break;
        }
    }

    system("pause");
    return 0;
}

当然，在主函数的开始我们就要使用struct addressbooks定义一个变量abs，这个abs是可以访问结构体内的成员的，否则那些函数可就没有实参咯！

以上就是这个项目的全部内容，内容比较简单，适合初学者，能帮助初学者更好的理解指针、数组和结构体的关系。
