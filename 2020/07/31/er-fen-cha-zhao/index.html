二分查找

二分查找的思想非常简单，相信许多小伙伴看一眼就理解了，但二分查找的细节很多，稍不注意就会出错，这篇文章就来讲一下常见的几个二分查找的情况。

二分查找V1.0

以下为模板V1.0，新手可以使用下面的模板：

二分查找的模板是固定的，要变的只是一些边界问题：

public int binarySearch(int[] nums, int target) {
    int left = 0, right = ...;
    while(...) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            ...
        } else if (nums[mid] < target) {
            left = ...
        } else if (nums[mid] > target) {
            right = ...
        }
    }
    return ...;
}

注意：计算 mid 时需要防止溢出， left + (right - left) / 2 有效防止了 因left 和 right 太大直接相加导致的溢出。

基本二分查找

此为最基本的二分查找算法，适用于数组中没有重复数字，或者是不要求寻找边界的情况：

public static int binarySearch(int[] nums, int target) {
    int low = 0;
    int high = nums.length - 1;
    while(low <= high){
        int mid = low + (high - low) / 2;
        if(nums[mid] == target){
            return mid;
        }else if(nums[mid] < target){
            low = mid + 1;
        }else{
            high = mid - 1;
        }
    }
    return -1;
}

查找左边界

这个算法适用于数组中有重复数字，且要求查找重复数字的左边界的情况，比如数组 [1, 2, 3, 4, 5, 8, 8, 8, 9, 10]，左边界的数组下标为 5，那么就返回 5。

public static int binarySearchLeft(int[] nums, int target) {
    int low = 0, high = nums.length - 1;
    while(low <= high){
        int mid = low + (high - low) / 2;
        if(nums[mid] > target){
            high = mid - 1;
        }else if(nums[mid] < target){
            low = mid + 1;
        }else{
            //此时 nums[mid] = target
            // nums[mid - 1] != target ：表示 nums[mid] 左边没有和 target 相等的数，即为左边界
            if(mid == 0 || nums[mid - 1] != target){
                return mid;
            }else{
                high = mid - 1;
            }
        }
    }
    return low;
}

查找右边界

这个算法适用于数组中有重复数字，且要求查找重复数字的右边界的情况，比如数组 [1, 2, 3, 4, 5, 8, 8, 8, 9, 10]，右边界的数组下标为 7，那么就返回 7。

public static int binarySearchRight(int[] nums, int target) {
    int low = 0, high = nums.length - 1;
    while(low <= high){
        int mid = low + (high - low) / 2;
        if(nums[mid] > target){
            high = mid - 1;
        }else if(nums[mid] < target){
            low = mid + 1;
        }else{
            //此时 nums[mid] = target
            // nums[mid + 1] != target ：表示 nums[mid] 右边没有和 target 相等的数，即为右边界
            if(mid == nums.length - 1 || nums[mid + 1] != target){
                return mid;
            }else{
                low = mid + 1;
            }
        }
    }
    return -1;
}

查找第一个大于等于给定值的元素

在有序数组中，查找第一个大于等于给定值的元素。比如，数组中存储的这样一个序列：3，4，6，7，10。如果查找第一个大于等于 5 的元素，那就是 6。


public int bsearch(int[] a, int n, int value) {
    int low = 0;
    int high = n - 1;
    while (low <= high) {
        int mid =  low + ((high - low) >> 1);
        if (a[mid] >= value) {
            if ((mid == 0) || (a[mid - 1] < value)) return mid;
            else high = mid - 1;
        } else {
            low = mid + 1;
        }
    }
    return -1;
}

查找最后一个小于等于给定值的元素

查找最后一个小于等于给定值的元素。比如，数组中存储了这样一组数据：3，5，6，8，9，10。最后一个小于等于 7 的元素就是 6。是不是有点类似上面那一种？实际上，实现思路也是一样的。


public int bsearch7(int[] a, int n, int value) {
    int low = 0;
    int high = n - 1;
    while (low <= high) {
        int mid =  low + ((high - low) >> 1);
        if (a[mid] > value) {
            high = mid - 1;
        } else {
            if ((mid == n - 1) || (a[mid + 1] > value)) return mid;
            else low = mid + 1;
        }
    }
    return -1;
}

搜索区间是左闭右闭还是左闭右开是由我们决定的，但一旦决定了，接下来的搜索区间就只能是这种形式（即左闭右闭或者是左闭右开）。

左闭右闭：

即 [left, right]。那么当我们发现索引 mid 不是要找的 target 时，如何确定下一步的搜索区间呢？

当然是去搜索 [left, mid - 1] 或者 [mid + 1, right] 对不对？因为 mid 已经搜索过，应该从搜索区间中去除。

左闭右开：

为什么 left = mid + 1，right = mid ？和之前的算法不一样？

答：这个很好解释，因为我们的「搜索区间」是 [left, right) 左闭右开，所以当 nums[mid] 被检测之后，下一步的搜索区间应该去掉 mid 分割成两个区间，即 [left, mid) 或 [mid + 1, right)。

还是那句话，一开始我们把搜索区间开闭情况定为什么，后面的子区间的开闭情况也必须是什么。

在左闭右开的情况里， [left, mid) 其实和 [left, mid - 1] 效果是相同的，只不过一个右边是打开的一个是闭合的而已。同样，因为两种情况下的左区间都是闭合的，所以左边该怎么变还怎么变。

二分查找V2.0

以下为模板V2.0，相对V1.0而言更难理解，但适用的范围更广，代码也相对简洁：

寻找左侧边界

public static int left_bound(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while(left < right){
        int mid = left + (right - left) / 2;
        if(nums[mid] >= target){
            right = mid;
        }else{
            left = mid + 1;
        }
    }
    return left;
}

用于数组中有重复元素查左边界的问题。

寻找右侧边界

public static int right_bound(int[] nums, int target) {
    if (nums.length == 0) return -1;
    int left = 0, right = nums.length - 1;
    while (left < right) {
        int mid = left + (right - left + 1) / 2;
        if (nums[mid] <= target) {
            left = mid;
        }else{
            right = mid - 1;
        }
    }
    return left;
}

用于数组中有重复元素查右边界的问题。

特别注意：一旦左边界 left = mid，mid 的值就必须为 left + (right - left + 1) / 2！

总结：

  • 二分查找的思想很简单，但一旦涉及到边界问题就会变得很复杂，想要练好二分查找算法，需要多刷题；

  • 推荐使用二分查找V2.0模板。
