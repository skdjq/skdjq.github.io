固定数组实现栈和队列

固定数组实现栈

public static class ArrayStack{
    private Integer[] arr;
    private Integer index;

    public ArrayStack(int initSize){
        if(initSize < 0){
            throw new IllegalArgumentException("The init size is less than 0!");
        }
        this.arr = new Integer[initSize];
        this.index = 0;
    }

    private Integer peek(){
        if(index == 0){
            throw new ArrayIndexOutOfBoundsException("the stack is empty!");
        }
        return arr[--index];
    }

    private void push(int num){
        if(index == arr.length){
            throw new ArrayIndexOutOfBoundsException("the stack is full!");
        }
        arr[index++] = num;
    }

    private Integer pop(Integer num){
        if(index == 0){
            throw new ArrayIndexOutOfBoundsException("the stack is empty!");
        }
        return arr[--index];
    }
}

固定数组实现队列

/**
     * end对应push，与size关联
     * start对应poll，与size关联
     * end与start解耦
     * 
*/
public static class ArrayQueue{
    private Integer[] arr;
    private Integer start;
    private Integer end;
    private Integer size;

    public ArrayQueue(int initSize){
        if(initSize < 0){
            throw new IllegalArgumentException("The init size is less than 0!");
        }
        arr = new Integer[initSize];
        start = 0;
        end = 0;
        size = 0;
    }

    private Integer peek(){
        if(size == 0){
            return null;
        }
        return arr[start];
    }

    private void push(int num){
        if(size == arr.length){
            throw new IndexOutOfBoundsException("the queue is full!");
        }
        size++;
        arr[end] = num;
        end = end == arr.length - 1 ? 0 : end + 1;
    }

    private Integer poll(){
        if(size == 0){
            throw new IndexOutOfBoundsException("the queue is empty!");
        }
        size--;
        int tmp = start;
        start = start == arr.length - 1 ? 0 : start + 1;
        return arr[tmp];
    }
}

实现特殊的栈

思路：

  • 使用两个栈，一个是正常的Data栈，一个是Min栈专门存放最小值。

  • 当push进Data栈时如果Min栈为空，同时将这个数push进Min栈。

  • 如果Min栈不为空，将push的这个数与Min栈栈顶比较，如果push的这个数更大，就再push一个Min栈栈顶元素进Min栈，否则将这个数push进Min栈。

  • 一旦pop，就将两个栈的栈顶元素都出栈。

public static class specialStack{

    private Stack<Integer> StackData;
    private Stack<Integer> StackMin;

    public specialStack(){
        this.StackData = new Stack<Integer>();
        this.StackMin = new Stack<Integer>();
    }

    private void push(int num){
        StackData.push(num);
        if(StackMin.isEmpty()){
            StackMin.push(num);
        }else{
            if(StackData.peek() < StackMin.peek()){
                StackMin.push(StackData.peek());
            }else{
                StackMin.push(StackMin.peek());
            }
        }
    }

    private Integer pop(){
        if(StackData.isEmpty()){
            throw new IndexOutOfBoundsException("the stack is empty!");
        }
        int res = StackData.pop();
        StackMin.pop();
        return res;
    }

    private Integer getMin(){
        if(StackMin.isEmpty()){
            throw new IndexOutOfBoundsException("the stack is empty!");
        }
        return StackMin.peek();
    }
}

public static void main(String[] args) {
    specialStack specialStack = new specialStack();
    for(int i = 1; i < 100; i++){
        specialStack.push(i);
    }
    System.out.println(specialStack.getMin());
}

队列结构实现栈结构

思路：

  • 使用两个队列，一个是Data队列，一个是Help队列。

  • 无论何时push，都只push到Data队列中。

  • 当需要pop时，将Data队列中的数依次poll到Help队列直至Data队列只剩下一个数。

  • 返回Data队列剩下的那一个数。

  • 两个队列引用交换，即：原来的Data变为Help，原来的Help变为Data。

public static class QueueToStack{
    private Queue<Integer> QueueData;
    private Queue<Integer> QueueHelp;

    public QueueToStack(){
        this.QueueData = new LinkedList<Integer>();
        this.QueueHelp = new LinkedList<Integer>();
    }

    private Integer peek(){
        if(QueueData.isEmpty()){
            throw new RuntimeException("the stack is empty!");
        }
        while(QueueData.size() > 1){
            QueueHelp.add(QueueData.poll());
        }
        int res = QueueData.poll();
        QueueHelp.add(res);
        swap();
        return res;
    }

    private void push(int num){
        QueueData.add(num);
    }

    private int pop(){
        if(QueueData.isEmpty()){
            throw new RuntimeException("the stack is empty!");
        }
        while(QueueData.size() > 1){
            QueueHelp.add(QueueData.poll());
        }
        int res = QueueData.poll();
        swap();
        return res;
    }

    // 交换两个队列的引用
    private void swap() {
        Queue<Integer> tmp = QueueData;
        QueueData = QueueHelp;
        QueueHelp = tmp;
    }
}

栈结构实现队列结构

思路和上面的使用队列实现栈差不多。

  • 准备两个栈：Push栈和Pop栈。

  • 所有的数据都push进Push栈，当要pop时，把Push栈里所有的元素都“倒入”Pop栈，然后返回Pop栈栈顶元素即可实现队列功能。

在Push栈的元素倒入Pop栈时有两个注意点：

  • Pop栈必须为空。

  • Push栈必须一次性将所有元素倒完。

public static class StackToQueue{
    private Stack<Integer> pushStack;
    private Stack<Integer> popStack;

    public StackToQueue(){
        pushStack = new Stack<Integer>();
        popStack = new Stack<Integer>();
    }

    public void push(int num){
        pushStack.push(num);
    }

    public int poll(){
        if(pushStack.isEmpty() && popStack.isEmpty()){
            throw new RuntimeException("The Queue is empty!");
        } else if (popStack.isEmpty()){
            while(!pushStack.isEmpty()){
                popStack.push(pushStack.pop());
            }
        }
        int res = popStack.pop();
        return res;
    }

    public int peek(){
        if (pushStack.isEmpty() && popStack.isEmpty()){
            throw new RuntimeException("The queue is empty!");
        }else if (popStack.isEmpty()){
            while(!pushStack.isEmpty()){
                popStack.push(pushStack.pop());
            }
        }
        int res = popStack.peek();
        return res;
    }

    private void swap() {
        Stack<Integer> tmp = pushStack;
        pushStack = popStack;
        popStack = tmp;
    }
}
