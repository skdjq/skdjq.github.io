дҪҝз”ЁCookie

дҫҝжҚ·ж–№ејҸпјҡ

  • дҪҝз”Ё repsonse.addCookie() ж–№жі•еҗ‘жөҸи§ҲеҷЁдҝқеӯҳ Cookie

  • дҪҝз”Ё request.getCookies() ж–№жі•иҺ·еҸ–жөҸи§ҲеҷЁеҪ’иҝҳзҡ„ Cookie

    • иҝ”еӣһзҡ„жҳҜ Cookie ж•°з»„

жЎҲдҫӢпјҡ

A.jsp дҝқеӯҳ CookieпјҢB.jsp иҺ·еҸ– CookieгҖӮ



    A


<%
    Cookie cookie1 = new Cookie("aaa","AAA");
    response.addCookie(cookie1);

    Cookie cookie2 = new Cookie("bbb","BBB");
    response.addCookie(cookie2);
%>

    Cookie[] cookies = request.getCookies();

        }
    }

Cookieзҡ„maxAge

Cookieзҡ„maxAgeвҖ”->Cookieзҡ„жңҖеӨ§з”ҹе‘ҪпјҢеҚіCookieеҸҜдҝқеӯҳзҡ„жңҖеӨ§ж—¶й•ҝпјҢд»Ҙз§’дёәеҚ•дҪҚгҖӮ

дҫӢеҰӮпјҡcookie.setMaxAge(60) иЎЁзӨәиҝҷдёӘCookieдјҡиў«жөҸи§ҲеҷЁдҝқеӯҳеҲ°зЎ¬зӣҳдёҠ60з§’

  • maxAge>0пјҡжөҸи§ҲеҷЁдјҡжҠҠCookieдҝқеӯҳеҲ°е®ўжҲ·жңәзЎ¬зӣҳдёҠпјҢжңүж•Ҳж—¶й•ҝдёәmaxAgeзҡ„еҖјеҶіе®ҡгҖӮ
  • maxAge<0пјҡCookieеҸӘеңЁжөҸи§ҲеҷЁеҶ…еӯҳдёӯеӯҳеңЁпјҢеҪ“з”ЁжҲ·е…ій—ӯжөҸи§ҲеҷЁж—¶пјҢжөҸи§ҲеҷЁиҝӣзЁӢз»“жқҹпјҢеҗҢж—¶Cookieд№ҹе°ұжӯ»дәЎдәҶгҖӮ
  • maxAge=0пјҡжөҸи§ҲеҷЁдјҡй©¬дёҠеҲ йҷӨиҝҷдёӘCookieпјҒ

Cookieзҡ„path

  • Cookieзҡ„pathе№¶дёҚжҳҜи®ҫзҪ®иҝҷдёӘCookieеңЁе®ўжҲ·з«Ҝзҡ„дҝқеӯҳи·Ҝеҫ„пјҒ
  • Cookieзҡ„pathз”ұжңҚеҠЎеҷЁеҲӣе»әCookieж—¶и®ҫзҪ®
  • еҪ“жөҸи§ҲеҷЁи®ҝй—®жңҚеҠЎеҷЁжҹҗдёӘи·Ҝеҫ„ж—¶пјҢйңҖиҰҒеҪ’иҝҳе“ӘдәӣCookieз»ҷжңҚеҠЎеҷЁе‘ўпјҹиҝҷз”ұCookieзҡ„pathеҶіе®ҡгҖӮ
  • Cookieзҡ„pathй»ҳи®ӨеҖјпјҡеҪ“еүҚи®ҝй—®и·Ҝеҫ„зҡ„зҲ¶и·Ҝеҫ„гҖӮдҫӢеҰӮеңЁи®ҝй—®/day11_1/jsps/a.jspж—¶пјҢе“Қеә”зҡ„cookieпјҢйӮЈд№ҲиҝҷдёӘcookieзҡ„й»ҳи®Өpathдёә/day11_1/jsps/
  • жөҸи§ҲеҷЁи®ҝй—®жңҚеҠЎеҷЁзҡ„и·Ҝеҫ„пјҢеҰӮжһңеҢ…еҗ«жҹҗдёӘCookieзҡ„и·Ҝеҫ„пјҢйӮЈд№Ҳе°ұдјҡеҪ’иҝҳиҝҷдёӘCookieгҖӮ

дҫӢеҰӮпјҡ

  • aCookie.path=/day11_1/;

  • bCookie.path=/day11_1/jsps/

  • cCookie.path=/day11_1/jsps/cookie/

    и®ҝй—®пјҡ/day11_1/index.jspж—¶пјҢеҪ’иҝҳпјҡaCookie

    и®ҝй—®пјҡ/day11_1/jsps/a.jspж—¶пјҢеҪ’иҝҳпјҡaCookieгҖҒbCookie

    и®ҝй—®пјҡ/day11_1/jsps/cookie/b.jspж—¶пјҢеҪ’иҝҳпјҡaCookieгҖҒbCookieгҖҒcCookie
